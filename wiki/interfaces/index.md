
# Interface
* [okf CLI](okf-cli.md) - The okf command: init, new, validate, search, read, index regen, serve.
* [okf-mcp](okf-mcp.md) - MCP server (FastMCP, stdio): search, read_concept, validate, create_concept, init_bundle + okf resources.
* [okf serve](okf-serve.md) - On-demand, read-only local web UI: tree, search, graph, reader.
