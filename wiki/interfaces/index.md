
# Interface
* [okf CLI](okf-cli.md) - The `okf` CLI — a thin argparse layer over the core plus code indexing and skill-only agent installation: init / new / validate / search / read / index / code / serve / agent install.
* [okf-mcp server](okf-mcp.md) - The `okf-mcp` MCP server — five tools (search/read_concept/validate/create_concept/init_bundle), per-concept okf:// resources, and the create_concept richness floor.
* [okf serve (web UI)](okf-serve.md) - A read-only, localhost, stdlib-only web UI over a bundle — tree, search, graph, and a Markdown reader with backlinks. Launched on demand, not by okf-mcp.

# Reference
* [okf:// resource scheme](okf-uri-scheme.md) - The okf://<bundle>/concepts/<id>.md MCP resource scheme — stable per-concept addressing alongside the read_concept tool.
