
# Interface
* [okf CLI](okf-cli.md) - The `okf` command — init, new, validate, search, read, index regen, serve.
* [okf-mcp](okf-mcp.md) - MCP server (FastMCP/stdio) exposing search, read_concept, validate + okf:// resources.
* [okf serve](okf-serve.md) - On-demand, read-only local web UI — tree, search, graph, reader (stdlib http.server + vanilla JS).
